Learning Objectives

  • Understand the main components of an IoT system
  • Experience the process of building an IoT system
  • Practice reflecting on the learning process
  • Learn to style web pages using flexbox and SASS/SCSS

Resources and Strategies

  • Self-study using online resources
  • Collaboration with classmates
  • Consultation with instructor
  • Documentation through photos and notes for reflection

Deliverables

  • IoT prototype: succulent monitoring app
  • Progress blog
  • Portfolio website

Assessment Criteria

  • Working IoT prototype
  • Evidence of reflection on the process
  • Use of flexbox and SASS/SCSS in the styling of the portfolio website
